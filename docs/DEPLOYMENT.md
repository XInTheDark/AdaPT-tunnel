# Deployment notes

The operator-facing setup flow is now documented primarily in:

- `README.md` — CLI-first quickstart and command overview
- `guides/DEPLOYMENT.md` — recommended guided setup flow
- `guides/MANUAL-CONFIG-SETUP.md` — raw config-file-oriented manual setup
- `guides/MANUAL-TESTING.md` — verification checklist

The older low-level deployment notes have been superseded by the user-friendly CLI flow built around:

- `apt-edge init`
- `apt-edge add-client`
- `apt-client import`
- `apt-edge start`
- `apt-client up`
